## 🔭 Мой текущий проект

### [Jamify](https://github.com/Nastydissonance/Jamify) (приватный репозиторий, доступ по запросу)

**Jamify** — веб-приложение для музыкантов, которые ищут единомышленников. Фильтрация по жанрам и инструментам, асинхронная загрузка данных, адаптивный дизайн в стилистике Discord.

**Стек:** HTML, CSS, JavaScript, json-server, Git, Figma (в планах).

**Почему приватный:** у проекта есть потенциал вырасти в коммерческий сервис. Я планирую развивать его и добавлять новые фичи. Код могу показать по запросу — пишите!

## 🛠️ Технологии и инструменты

- **Frontend:** HTML5, CSS3 (Flexbox, Grid, адаптив), JavaScript (ES6+)
- **Асинхронность:** fetch, async/await, обработка ошибок, лоадеры
- **Бэкенд (для тестов):** json-server
- **Система контроля версий:** Git, GitHub (приватные/публичные репозитории)
- **ОС и окружение:** Arch Linux + Hyprland (стараюсь настраивать сам, включая waybar, rofi, swww/awww)
- **Редактор:** Neovim (LazyVim)
